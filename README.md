# Frequency-Calculator-Basic
Basic frequency calculator based on python 3 

# How to use
Windows
    install python from the link:https://www.python.org/downloads/windows/ \n
    download the Frequency-Calculator-win.py file \n
    open file and enter the  elements one by one \n
    type end in lowercase to stop entering elements. \n
    the element and its frequency will be displayed in a table \n
