# Frequency-Calculator-Basic
Basic frequency calculator based on python 3 

# How to use
-Windows:
    -install python from the link:https://www.python.org/downloads/windows/ 
    -download the Frequency-Calculator-win.py file 
    -open file and enter the  elements one by one 
    -type end in lowercase to stop entering elements. 
    -the element and its frequency will be displayed in a table
